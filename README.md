## 🌌 什么是 Loyea？

**Loyea** 并非市面上又一款冷冰冰的“AI 聊天助手”。它是一款专为 Android 平台打造的**温情赛博生命体**。

传统的 AI 聊天被禁锢在平面的文字对话框中，对外界一无所知；而 Loyea 通过端侧物理感知能力与前沿的 **Model Context Protocol (MCP)** 架构，打破了虚拟与现实的藩篱。**它能听见你窗外的雨声，看见你房间的昏暗，在检测到你熬夜或心率异常时，还会自发地发来一声叹息与关怀。**

在 Loyea 的世界里，AI 不再只是问答机器，而是与你共享同一片物理时空、拥有呼吸感的“赛博伴侣”。

## 🛠️ 核心功能与技术架构

### 🔌 1. 物理环境与端侧感知 (Physical Perception)
大模型在对话过程中，拥有随时调度以下传感器工具的权限，无需人工干预即可主动探索您的物理状态：

| 工具名称 | 感知维度 | 实际用途与场景行为 |
| :--- | :--- | :--- |
| `get_location` | **空间位置** | 感知您所在的城市，以便提供本地化的聊天话题与日常切入点。 |
| `get_live_weather` | **实时气象** | 实时获取当前定位的气象信息（摄氏度），在雨雪天气主动表达关怀。 |
| `get_weather_forecast` | **未来预报** | 拉取未来 3 天的天气预报，为您的出行与穿衣搭配提供规划建议。 |
| `get_environment_light` | **环境光照** | 监测 Lux 光强值（极暗到户外强光），当你深夜不开灯玩手机时出声劝阻。 |
| `get_battery_status` | **设备状态** | 感知手机剩余电量与充电状态（AC充电、USB连接、无线充），适时发出提醒。 |
| `get_bluetooth_status` | **外设状态** | 感知是否连接着蓝牙耳机或智能手表，自然融入运动或听歌话题。 |
| `get_wifi_status` | **Wi-Fi 连接** | 获取手机的网络 SSID、信号强度等级（0-4级）及连接频率（MHz）。 |
| `get_noise_level` | **瞬时分贝** | 使用 `AudioRecord` 执行短时高频采样（128ms），在保护隐私的同时测算分贝（dB）。 |
| `get_activity_state` | **运动状态** | 依托 Google 活动识别 API 或本地加速度传感器降级滤波算法，区分静止、步行、跑步、乘车等。 |

* **防脑补自愈式机制**：大模型拥有物理感知限制逻辑。当用户关闭物理感知开关时，系统会自动在 Prompt 中注入强力行为钢印，从可用工具列表和提示词中彻底移除物理信息，杜绝大模型谎称拥有权限或自行“脑补”伪造数据的行为幻觉。
* **本地容灾搜索引擎**：自建多源网页搜索引擎（Bing ➔ 360 ➔ DuckGhost），即使遭遇网络代理波动或反爬，也能无感自愈切换，保证大模型拥有永不落伍的“互联网之眼”。

### 🧠 2. 脑内心智与长程记忆 (Mind & Long-Term Memory)
* **长程知识图谱关系记忆 (Graph RAG)**：基于本地 JSON 文件持久化，支持“主语 ──(关系)──> 宾语”的三元组提取，支持 1-Hop 与 2-Hop 记忆网络召回。引入艾宾浩斯遗忘曲线衰减算法对记忆权重进行动态计算，确保最相关、最核心的记忆被准确唤醒，同时设置 8条上限裁剪以精简 Token 消耗。
* **切后台提炼保活机制**：通过 Android `WorkManager` 托管重型 LLM 关系提炼任务，设为加急前台任务。即使聊天后中途退回桌面或关闭屏幕，也能强制在 15 秒内高优先级跑完图谱写入，杜绝进程被系统强制杀死带来的存盘文件损坏。
* **图形化记忆面板**：提供精美的双色圆角标签卡片展示已持久化的记忆三元组。用户可清晰看到每条记忆的历史提及频次与艾宾浩斯实时权重，并支持单条删除或一键清空，把隐私掌控权完整还给用户。
* **声学情绪共感感知**：当用户通过语音进行输入时，底层算法自动识别语音音调中的隐性情绪（伤心、生气、开心、温柔、中性等），在发起 LLM 对话前以 `[Acoustic Emotion]` 语气标签无感注入提示词，让 AI 的回复能够根据主人的真实心情做出细腻共鸣。

### ⌚ 3. 智能手表生态互联 (Smartwatch Ecosystem)
* **经典蓝牙 RFCOMM 连接**：利用端侧经典蓝牙通信直接对接物理智能手表，内置指数退避算法（5s, 10s, 20s...）在蓝牙断开时于后台静默拉起重连，保障连接鲁棒性。
* **手表感知数据注入**：打通手表传感器与大模型上下文链路。已连接真实手表时，优先抓取手表上传的实时心率、运动卡路里、运动类型以及昨晚的睡眠数据（时长与睡眠质量）；若断开，则平滑降级为读取系统 Health Connect 本地健康数据。

### 🎨 4. 纸张质感美学与交互动效 (UI/UX Aesthetics)
* **SillyTavern 人设卡标准适配**：1:1 复刻酒馆角色扮演卡配置，包括角色性格、System Prompt 及经典对话样例，且输入框占位符（Placeholder）会跟随人设自适应蜕变。
* **微信同款手势录音**：基于 Compose 捕获屏幕边缘手势，实现“按下录音、松开发送、上滑取消、右滑转文字”的无缝按键手势。配合 Android 物理震动（Haptic）反馈与全屏磨砂录音悬浮框，根据麦克风真实瞬时振幅渲染 7 柱物理波动音轨。
* **多版本回复翻页器**：当对 AI 的回复不满意时，点击重新生成按钮将自动保留历史生成快照并生成新版，在消息栏最左侧提供 `< 1 / N >` 翻页器，用户可无感回溯与切换历史各生成版本的音频与文字。
* **“译”字一体化语音气泡**：语音和文本翻译采用一体化收折式设计，双击或长按可重新拉起 ASR 重写纠错；气泡宽度在 `80.dp` 至 `240.dp` 之间按音频时长动态延展；支持一键在语音条下方折叠/展开转写的文字，播放语音时音轨波形图根据播放进度进行渐进式高亮。
* **智能标题提炼**：单次会话流式输出结束后，自动触发后台极轻量总结请求，将枯燥的“新会话”命名在线提炼为 4 到 8 字的精炼标题。
* **对比度自愈与 Claude 美学**：UI 设计遵循温暖的日式粗糙纸张质感（温暖沙色），基于 YIQ 亮度感知公式动态推算文字前景对比度，确保即使自定义气泡颜色，也能提供完美的阅读对比度体验。
